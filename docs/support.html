Open a GitHub issue

Before Filing an Issue

  1. Confirm your Mac is running macOS 13 or newer.
  2. Connect the Android device by USB and enable USB debugging, or pair through Android Wireless debugging.
  3. Open PhoneRelay's log file from the Help menu and include the relevant error lines.
  4. Include the app version, macOS version, Android version, connection type, and whether mirroring worked over USB.
